# 文本分类标注功能

## 概述

文本分类标注功能允许用户对文本内容进行分类标注，通过动态表单的方式收集分类信息。

## 功能特性

### 1. 动态表单支持
- **单选框 (Radio)**: 支持单选选项，包括"其他"选项的自定义输入
- **多选框 (Checkbox)**: 支持多选选项
- **输入框 (Input)**: 支持文本输入，可设置最大长度和占位符
- **文本域 (Textarea)**: 支持多行文本输入
- **下拉选择 (Select)**: 支持下拉选择

### 2. 表单验证
- 支持必填字段验证
- 提交时自动检查必填项

### 3. 自动保存
- 每30秒自动保存表单数据
- 显示最后保存时间

### 4. 键盘快捷键
- `Ctrl/Cmd + S`: 手动保存
- `Ctrl/Cmd + Enter`: 提交分类结果
- `Tab`: 在表单字段间切换
- `Space`: 选择/取消选择当前选项
- `Esc`: 取消当前操作

## 使用方法

### 1. 访问页面
访问 `/editor/text/Second?type=classification` 进入文本分类标注页面

### 2. 查看文本
左侧显示需要分类的文本内容，支持滚动查看

### 3. 填写分类表单
右侧显示动态生成的分类表单：
- 根据接口返回的字段配置动态渲染不同类型的表单项
- 必填字段标有红色星号 (*)
- 根据字段类型选择相应的输入方式

### 4. 保存和提交
- 系统每30秒自动保存
- 可使用快捷键 `Ctrl/Cmd + S` 手动保存
- 填写完成后点击"提交"按钮或使用 `Ctrl/Cmd + Enter` 提交

## 技术实现

### 组件结构
```
Class/
├── index.tsx                    # 主页面组件
├── types.ts                     # 类型定义
├── store.ts                     # Zustand状态管理
├── components/
│   ├── ClassificationHeader/    # 页面头部组件
│   ├── TextDisplay/            # 文本显示组件
│   └── DynamicForm/            # 动态表单组件
└── hooks/
    └── useKeyboardShortcuts.ts  # 键盘快捷键Hook
```

### 状态管理
使用 Zustand 进行状态管理，包括：
- 文本内容和表单配置数据
- 表单值状态
- 加载和错误状态
- 保存和提交操作

### 样式设计
- 响应式布局，支持移动端适配
- 左右分栏设计，左侧文本，右侧表单
- 统一的视觉风格和交互体验

## 扩展性

### 添加新的表单字段类型
1. 在 `types.ts` 中扩展 `FormFieldType` 类型
2. 在 `DynamicForm/index.tsx` 中添加对应的渲染逻辑
3. 在样式文件中添加相应的样式

### 自定义验证规则
可以在 `store.ts` 的 `submitClassification` 方法中添加自定义验证逻辑

### 接口集成
目前使用模拟数据，可以在 `store.ts` 中替换为真实的API调用
